DOM(Document Object Model) : 문서(html) 객체 모델

  • '' 하위 태그는 각각의 기능(Method), 속성(properties)를 갖고 있는 문서 객체이다.
  • 엘리먼츠 가져오기 : tag element로 가져오는 메서드 ById, ByClassName, ByTagName
  • document.getElementById('idName') : 단일 엘리먼츠 리턴
  • document.getElementByClassName('className') : 복수 엘리먼츠 리턴 => HTMLCollection 타입으로 리턴
  • document.getElementByTagName('tagName') : 복수 엘리먼츠 리턴 => HTMLCollection 타입으로 리턴
  • getElement로 가져오는 객체는 우선적으로 HTMLCollection의 index를 선택해야 함
