변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 실수, NaN(not a number : 숫자가 아님이라는 값)
  • 문자 : String => "", '', ``으로 감싼 값들
  • 불리언 : Boolean => true/false
  • 객체 : Object => 1개 이상의 데이터가 특정한 구조를 가지고 있는 형태 (아무것도 없는 사실도 1개로 취급) {}
  • 배열 : Array => [1,2,3 ...] 형태로 이루어진 데이터 집합 (순서, 길이 있음.)
  • 알 수 없는 값 : null => 뭔가 있긴한데 파악이 안됨
  • 미정된 값 : undefined : 값이 할당되지 않음
  • 데이터 타입 확인 명령어 : typeof 변수 / 값

변수의 관리

  1. 기본 자료형(primative Type) : 값이 메모리의 주소값이 가리키는 곳에 직접 할당
    number, boolean, string(원래는 참조자료형이지만 사용편의성을 위해 같이 사용가능)
  2. 참조자료형(Reference Type) : 값이 저장되어 있는 메모리 주소값을 참조하고 있는 할당
    string, object, array, funtion, null, undefined
