이벤트 객체는 전역객체로써 모든 태그에 객체를 참조시킬 수 있다.

  • a 태그와 button 태그에 이벤트를 주로 참조시켜 클릭에 대한 반응을 기대할 수 있게 만드는 구조가 일반적임
  • a태그는 href 속성에 의해 자체 이동이 구현되어 있어서 이동을 막는 코드가 필요
  • 이벤트 객체를 부여하는 방식 3가지. 각자 목적과 작동방식이 다르다.
  • 화면 출력 영역 document영역(객체)라고 한다.
  • 태그는 html element, 속성은 attribute, 일반 텍스트는 textNode라고 한다.
  • html 태그로 출력 : innerHTML라는 property를 사용
  • text로 출력 : innerText라는 property를 사용
  • 객체의 값을 가져올 때 (id) : getElementById('id명')
  • 객체의 값을 가져올 때 (class) : querySelector('selector')
